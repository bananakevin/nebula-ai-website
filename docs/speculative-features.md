## 📡 API 規格(待官方文件發布)

目前**沒有公開的 API 文件**。以下是基於常見 AI 平台推測的可能端點:

### 可能的 API 端點
```
POST   /v1/agents/create          # 建立 agent
POST   /v1/agents/{id}/tasks      # 指派任務
GET    /v1/tasks/{id}/status      # 查詢任務狀態
GET    /v1/agents/{id}            # 取得 agent 資訊
DELETE /v1/agents/{id}            # 刪除 agent
GET    /v1/tools                  # 列出可用工具
```

### 可能的錯誤碼
- `400 Bad Request` - 請求格式錯誤
- `401 Unauthorized` - 驗證失敗
- `403 Forbidden` - 權限不足
- `404 Not Found` - 資源不存在
- `429 Too Many Requests` - 超過速率限制
- `500 Internal Server Error` - 伺服器錯誤
- `503 Service Unavailable` - 服務暫時無法使用

⚠️ **這些端點和錯誤碼是推測的,實際 API 請等待官方文件**
